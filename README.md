# PR2S 
Public Room Reserve System
공공기관 공실 대여 시스템
<br>
<br>
<br>

## 📃 프로젝트 개요
공공기관의 공실을 효율적으로 관리할 수 있는 대여·관리 시스템
<br>
<br>
<br>
## 🚪 공공기관 공실 대여 시스템이란
대여할 공간의 시간표를 확인하고 비어있는 시간을 찾아 대여해야 했던 기존의 스케줄러들과는 다르게,   
누구나 쉽고 간단하게 공실을 찾아 대여할 수 있는 시스템
<br>
<br>
<br>
